1、远程建立仓库
	
2、克隆仓库到本地
	git clone
3、新建分支
	git checkout -b dev
4、推送到远程分支
	git push origin dev:dev
5、版本回退
	查看提交：git reflog
	回退版本：git reset --hard 版本号


git客户端操作步骤
	1、克隆仓库的本地
		git clone 地址
	2、建立本地分支
		git checkout -b 本地分支名
	3、切换到本地分支（push到仓库后要回到本地分支）
		git checkout 本地分支的名字
	4、 本地提交
		git add.
		git commit -m ‘提交内容’
	5、切换到远程dev分支
		git checkout dev
	6、与dev分支合并
		git merge dev
	7、提交到远程分支
		git push origin dev
	8、切换到本地分支
		git checkout 本地分支名称
